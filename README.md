# 📚 Lógica de Programação

Este repositório reúne os exercícios desenvolvidos durante meus estudos de Lógica de Programação.

O curso foi originalmente proposto utilizando Visualg, porém optei por implementar os exercícios em JavaScript para consolidar os conceitos de lógica em uma linguagem com a qual já possuía familiaridade. Posteriormente, os mesmos exercícios foram organizados para servir como base de comparação entre diferentes linguagens de programação.

## 🎯 Objetivos

- Desenvolver o raciocínio lógico e a capacidade de resolução de problemas.
- Praticar estruturas fundamentais da programação.
- Comparar a implementação dos mesmos algoritmos em diferentes linguagens.
- Documentar meu processo de aprendizado.

## 📂 Estrutura do repositório

```text
logica-programacao/
```
